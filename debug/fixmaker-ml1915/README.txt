
Notes
-------------
The Dell 1130 printer has the Samsung ML-1915 chipset installed.
(See the docs folder for more information and pictures)


Requirements
-------------
- Windows 7
- Required libraries are included (./lib folder)


Usage
-------------

1. Disconnect ALL USB devices except HID devices (like mouse, keyboard) and your driver.
2. Turn on your Dell 1130 printer.
2. Drag the OS image on the usbprns2.exe program.
3. Alternatively, run usbprns2.exe ML1910_V1.01.00.83.fls in Command Line.
